# Barnaby
Barnaby is a personal assistant.

## Examples
* "How's the weather?"
* "How's the weather in Lisbon?"
* "Show me the news"

## Prequesites
* Python 3
* nltk (Module)
* Numpy (Module)
* Random (Module)
* String (Module)
* SKLearn (Module)
* Requests (Module)
* SpeechRecognition (Module)
* pyttsx3 (Module)
* espeak (Module) in Linux
* nsss (Module) in macOS
* sapi5 (Module) in Windows
